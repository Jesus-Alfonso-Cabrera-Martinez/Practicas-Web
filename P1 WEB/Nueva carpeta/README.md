# Asignación 1 - Fundamentos de Laravel

Este paquete contiene los archivos mínimos para la Asignación 1:
- routes/web.php
- app/Http/Controllers/PaginaController.php
- resources/views/bienvenida.blade.php
- resources/views/saludo.blade.php
- .gitignore

Instrucciones:
1. Coloca estos archivos en la estructura de un proyecto Laravel existente o crea uno nuevo con `composer create-project laravel/laravel asignacion1`.
2. Copia los archivos a las rutas indicadas.
3. Ejecuta `php artisan serve` y visita:
   - /bienvenida
   - /saludo/TuNombre
